### Description

#

### How to install:

1. Go to GitHub.com and find GitHub profile of 21stCenturyCowboy94.

1. Find the link of repo of "independent_portfolio."

1. Find the SSH link of the repo and copy it.

1. Paste SSH link in terminal after typing the words "git clone."

1. Once in the terminal storage, use "ls" command to find file, and use "cd" command to go to file.

1. Type in command "code ." and it should pull up in shell command in VS Code page.

1. Find the "Go Live" button in the bottom right corner, and click it. (Requires Go Live extension in VS Code)
